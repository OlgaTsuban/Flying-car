# Flying-car
<p>Here are tips how to play the game</p>
<p> 1. Clone this repository: git clone https://github.com/OlgaTsuban/Flying-car.git</p>
<p> 2. Compile the program using: make</p>
<p> 3. Start the game: ./endgame</p>
